# Filipino Mahjong: Joker Rules

Authentic Filipino Mahjong for Android with Joker rules — the way you play at home with your family.

Built by Bayanihan Games.

## Website
[filipinomj.com](https://filipinomj.com)

## About
Filipino Mahjong: Joker Rules is a single-player Android game featuring authentic Filipino Mahjong rules including Joker tiles, Bisaklat, Búnot, and Siete Pares.

Built with HTML/JavaScript wrapped in Capacitor for Android.
